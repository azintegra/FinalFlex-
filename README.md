# Flex Gate & Locker Codes

Simple, fast, mobile-optimized lookup tool for gate and locker codes.

## Deployment

1. Create a new GitHub repository
2. Upload:
   - index.html
   - data.json
3. Go to Settings → Pages
4. Select Deploy from branch → main → /root
5. Save

Your app will be live in seconds.
